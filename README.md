# Homebrew Tap for Ferrite

## Install

```bash
brew tap olaproeis/ferrite
brew install --cask ferrite
```

## Update

```bash
brew upgrade --cask ferrite
```

## About

[Ferrite](https://github.com/OlaProeis/Ferrite) is a fast, lightweight markdown editor built in Rust with egui.

This tap provides a Homebrew Cask that installs Ferrite.app to /Applications
and automatically handles macOS Gatekeeper (quarantine attribute removal).
